## How to Read IEEE Citations

**Books**: [#] Author(s), *Title*, edition. City, State/Country: Publisher, Year.

**Journal Articles**: [#] Author(s), "Article title," *Journal Name*, vol. X, no. Y, pp. Z–W, Month Year.

**Conference Papers**: [#] Author(s), "Paper title," in *Proc. Conference Name (ABBREV)*, Year, pp. Z–W.

**Online Resources**: [#] Author/Organization, "Title," Year. [Online]. Available: URL

---

## Using This Bibliography

- **In-text citations**: Use bracketed numbers [1], [2] matching entries above
- **Multiple citations**: [1], [3], [5] or [7]–[10] for ranges
- **Direct quotes**: Add page numbers: [2, p. 45]
- **Paraphrasing**: Cite source without page number: [5]

**Contributing**: When adding new chapters, insert citations in numerical order of first textbook appearance. Update this file accordingly.
